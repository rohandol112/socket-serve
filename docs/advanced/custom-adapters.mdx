## Adapter Interface

Adapters integrate socket-serve with frameworks.

## Requirements

An adapter must:
1. Handle GET/POST requests
2. Manage session lifecycle
3. Serve client SDK

## Reference Implementation

See `src/adapters/express.ts` and `src/adapters/nextjs.ts` for reference implementations.
